# AegisLock Site

Product website for AegisLock.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push to `main`. GitHub Pages builds `dist` through `.github/workflows/pages.yml`.

## Positioning

AegisLock is positioned as honest private app protection for an unlocked Mac:

- Safe Hide by default.
- Touch ID or Mac password.
- Local Access Reports.
- Agentic Work leases.
- Clear security boundary. No impossible admin/root/malware claims.
