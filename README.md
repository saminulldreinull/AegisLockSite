# AegisLock Site

Astro product website for AegisLock.

## Run

```bash
npm install
npm run dev
```

Local dev serves the GitHub Pages base path:

```bash
http://127.0.0.1:4321/AegisLockSite/
```

## Build

```bash
npm run build
```

## Deploy

Push to `main`. GitHub Pages builds Astro output in `dist` through `.github/workflows/pages.yml`.

## Positioning

AegisLock is positioned as honest private app protection for an unlocked Mac:

- Safe Hide by default.
- Touch ID or Mac password.
- Focus-based relock option.
- Local Access Reports.
- Agentic Work leases.
- Clear security boundary. No impossible admin/root/malware claims.
